# 🏨 Hotel Management System (HMS)

## Structure
```
hms/
├── backend/    Node.js + Express  → http://localhost:3000
├── frontend/   React + Vite       → http://localhost:5173
└── database/   SQL migrations + seeds
```

## Start Backend
```bash
cd backend && npm run dev
```

## Start Frontend
```bash
cd frontend && npm run dev
```
